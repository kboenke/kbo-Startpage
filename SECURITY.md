# Security Policy

## Supported Versions

Only the latest version published on either of the Add-On stores is supported:

| Version | Supported          |
| ------- | ------------------ |
| 1.8.0   | :white_check_mark: |
| 1.7.1   | :white_check_mark: |
| < 1.7   | :x:                |

## Reporting a Vulnerability

Please open an issue on this repository.
