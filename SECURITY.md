# Security Policy

## Supported Versions
Only the most recent version published in the browser add-on stores is actively supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.9.3   | :white_check_mark: |
| 1.9.2   | :x:                |
| 1.9.1   | :white_check_mark: |
| 1.9.0   | :x:                |
| 1.8.1   | :x:                |
| 1.8.0   | :x:                |
| 1.7.1   | :x:                |
| < 1.7   | :x:                |

## Reporting a Vulnerability
To report a security vulnerability, please open an issue in this repository.
